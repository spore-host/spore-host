What spore.host Is

spore.host is an open-source command-line tool suite and web dashboard for launching and managing AWS EC2 instances. The Service:

SMS Notifications

spore.host offers optional SMS notifications for instance lifecycle events (TTL warnings, idle stops, job completions). Message frequency varies based on instance activity. Message and data rates may apply.

  • Opt-in: Run spawn defaults set phone +1XXXXXXXXXX in the CLI. No messages are sent until you explicitly register your number.
  • Opt-out: Reply STOP to any message at any time, or run spawn defaults unset phone.
  • Help: Reply HELP to any message for support information.
  • SMS notifications are operational only — no marketing messages are sent.
  • Carriers are not liable for delayed or undelivered messages.
